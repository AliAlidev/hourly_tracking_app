# Image Assets

Place your gallery images in this directory.

## Required Images

The app expects 12-20 images with the following names:

- `nature_1.jpg`
- `nature_2.jpg`
- `city_1.jpg`
- `city_2.jpg`
- `abstract_1.jpg`
- `abstract_2.jpg`
- `landscape_1.jpg`
- `landscape_2.jpg`
- `architecture_1.jpg`
- `architecture_2.jpg`
- `sunset_1.jpg`
- `sunset_2.jpg`
- `mountain_1.jpg`
- `mountain_2.jpg`
- `ocean_1.jpg`
- `ocean_2.jpg`
- `forest_1.jpg`
- `forest_2.jpg`

## Image Requirements

- Format: JPG or PNG
- Recommended size: 1080x1080px or larger
- File size: Keep under 2MB per image for optimal performance
- Content: Nature, cities, abstract art, landscapes, architecture, etc.

## Adding Your Own Images

1. Add your images to this directory
2. Update the `_imagePaths` list in `lib/main.dart` to match your image filenames
3. Ensure all images are properly referenced in `pubspec.yaml` (already configured for `assets/images/`)
